# python-programs
A complete collection of Python programs from basic to advanced with simple explanations, coding exercises, and mini projects.

🐍 Python Counting Programs
Welcome to the Python Counting Programs repository.
This repository contains different ways to print counting in Python, starting from the most basic approach to slightly advanced concepts. It is specially designed for beginners who are learning Python step by step.

📂 Programs Included
1. Counting Using Print Statements
Print numbers manually using multiple print() statements.
2. Counting Using Variables
Store numbers inside variables and print them.
3. Counting Using While Loop
Print counting using a while loop.
4. Counting Using For Loop
Generate counting using the range() function.
5. Counting Using User Input
Take the ending number from the user and print the counting.
6. Counting Using Functions
Create a function to print counting.
7. Counting Using Function Parameters
Pass starting and ending values as parameters.
8. Custom Range Counting
Take both starting and ending numbers from the user.
9. Negative to Positive Counting
Print numbers from -100 to 100.
10. Reverse Counting
Print numbers in reverse order.
